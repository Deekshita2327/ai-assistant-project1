# SmartQuery – AI-Powered Q&A System

## Live Demo
Frontend: https://ai-assistant-project-gamma.vercel.app  
Backend: https://ai-assistant-project-m2dq.onrender.com  

## Features
- Ask questions using AI  
- Responses in bullet-point format  
- Full-stack integration (React + Node.js + MongoDB)  
- Deployed using Vercel and Render  

## Tech Stack
- Frontend: React (Vite)  
- Backend: Node.js, Express.js  
- Database: MongoDB Atlas  
- AI: Groq API  

## Description
SmartQuery is a full-stack conversational AI web application that allows users to ask questions and receive structured responses. The system integrates a React frontend with a Node.js backend and communicates with the Groq AI API.
